# SDF Primitives Demo

Visual showcase of every SDF primitive in a grid layout.

```
arcane dev demos/sdf-primitives/sdf-primitives.ts
```

Displays ~20 shapes arranged in rows:
- Row 1: circle, box, rounded_box, ellipse, segment
- Row 2: triangle, egg, heart, moon, hexagon
- Row 3: pentagon, star(5), star(6), cross, ring

Each shape rendered in a distinct color with label text.
