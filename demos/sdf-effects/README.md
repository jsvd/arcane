# SDF Effects Demo

Visual effects showcase demonstrating animated SDF capabilities.

```
arcane dev demos/sdf-effects/sdf-effects.ts
```

Scene layout (3x3 grid):
- Top row: Cosine palette, gradient mountains, glow heart
- Middle row: Pulsing star, rotated star, scaled glow circle
- Bottom row: Repeat pattern (tiled), mirror symmetry, onion rings

Each effect labeled with text. Demonstrates fill types and domain transforms.
