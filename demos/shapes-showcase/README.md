# Shapes Showcase

Demonstrates all geometry-pipeline shape primitives added in Phase 26.

## Shapes Shown

- **Filled**: `drawCircle`, `drawEllipse`, `drawTriangle`, `drawSector`
- **Outlines**: `drawArc`, `drawRing`
- **Lines**: `drawLine` (varying thickness), `drawCapsule`
- **Polygons**: `drawPolygon` (hexagon, star)

All shapes animate (pulse, rotate) to show GPU-rendered smoothness.

## Running

```bash
arcane dev demos/shapes-showcase/shapes-showcase.ts
```
